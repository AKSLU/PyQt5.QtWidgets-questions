# PyQt5.QtWidgets-questions

Простой тестовый конструктор на PyQt5 с вопросами и вариантами ответов. Результат сохраняется в файл `result.txt`.

---

## Требования

* Python 3.10+
* `PyQt5`

Установите библиотеку:

```bash
pip install PyQt5
```

---

## Структура проекта

* `5.py` — основной скрипт приложения.

---

## Использование

1. Клонируйте репозиторий:

```bash
git clone https://github.com/AKSLU/PyQt5.QtWidgets-questions.git
cd PyQt5.QtWidgets-questions
```

2. Запустите тест:

```bash
python 5.py
```

3. Выберите правильный вариант ответа на каждый вопрос и нажимайте кнопку `Дальше`. После последнего вопроса нажмите `Завершить`, чтобы увидеть результат.

---

## Особенности

* Поддержка нескольких вопросов с тремя вариантами ответов.
* Проверка правильного выбора.
* Отображение предупреждения, если ответ не выбран.
* Сохранение результата в файл `result.txt`.
* Простая графическая реализация на PyQt5.
